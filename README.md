#Название проекта: Генератор случайных предложений
Суть этой программы заключается в автоматизации творческого процесса создания случайных фраз. Программа работает как мини-движок генерации контента, который собирает готовые «кирпичики» (слова) в осмысленные предложения по заданному шаблону.
Вот логика работы программы, разбитая на ключевые этапы:
1. Архитектура данных (База знаний)
В основе лежат три списка слов: подлежащие, сказуемые и объекты. Это «словарь» программы. Вместо того чтобы писать сотни предложений вручную, программа хранит базовые компоненты, из которых можно собрать тысячи уникальных комбинаций.
2. Алгоритм «Случайного выбора»
Когда пользователь запускает генерацию, программа обращается к библиотеке random. Используя функцию choice, она делает независимый выбор по одному слову из каждого списка. Поскольку каждый выбор случаен, вероятность того, что программа выдаст одно и то же предложение дважды, крайне мала.
Запуск: python main.py

Как работает программа?
1. Подключение инструментов:
import random: Загружает библиотеку для работы со случайностью.
import time: Загружает библиотеку для работы со временем (нужна для time.sleep).
2. Данные (Глобальные константы):
SUBJECTS, VERBS, OBJECTS: Это списки слов. Написание имен переменных заглавными буквами — это стандарт программирования для констант (данных, которые не меняются в процессе работы).
3. Функция generate_sentence:
return f"{random.choice(...)} ...": Здесь используется random.choice для каждого списка и сразу «склеиваешь» их в одну строку. Это очень лаконично и профессионально.
4. Функция main (Основная логика):
while True:: Создает «бесконечный цикл». Программа будет работать, пока пользователь сам не выберет пункт «2. Выход» (команда break).
5. try ... except Exception as e: Если что-то пойдет не так (например, если программа столкнется с непредвиденной системной ошибкой).То программа защитится от «падения», 
6. time.sleep(0.5): Создает искусственную задержку в полсекунды. Это улучшает UX (User Experience) — делает работу программы «живой» и менее резкой для глаз пользователя.
7. Точка входа:
if __name__ == "__main__":: Гарантирует, что функция main() запустится, только если ты запустил этот файл напрямую (python main.py), а не импортировал его как модуль в другой проект.
